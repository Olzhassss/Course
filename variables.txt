This document contains essential custom PHP variables used in this project.

	General. Folders and addresses:

> PAGE URL - $url
> CURRENT PAGE URI - $thispage

> 'ROOT' FILE ADDRESS - $root_uri
> ROOT DIRECTORY PREFIX - $temp
> MAIN PAGE ADDRESS - $index

> CUSTOM SCRIPTS FOLDER ADDRESS (url) - $js
> CUSTOM STYLESHEETS FOLDER ADDRESS (url) - $stylesheets
> IMAGES FOLDER ADDRESS (url) - $imgs

> HEAD TAG FILE ADDRESS - $head_uri
> HEADER FILE ADDRESS - $header_uri
> SPECIAL HEADER FILE ADDRESS - $headerAdmin_uri 
> FOOTER FILE ADDRESS - $footer_uri
> 'BACK' LINK FILED ADDRESS - $back_link_uri

	General. Scripts and shylesheets:

> CUSTOM SCRIPTS ARRAY (files' names) - $custom_scripts
> CUSTOM STYLESHEETS ARRAY (files' names) - $custom_stylesheets
> CUSTOM STYLES (CSS) - $custom_styles

	Specific. References:

> PRE-APPLICATION PAGE ADDRESS- $app0;
> TEACHERS APPLICATION FORM HREF - $app_tch
> STUDENTS APPLICATION PAGE HREF - $app_std0
> STUDENT'S PRIVATE LESSONS APPLICATION FORM HREF - $app_std_private
> STUDENT'S GROUP LESSONS APPLICATION FORM HREF - $app_std_group
> ADMINISTRATOR LOG IN PAGE ADDRESS - $authorization_href

> TOP (HEADER) SECTION HREF - $header_href
> CONTACTS SECTION HREF - $contacts_href
> PRICE LIST SECTION HREF - $pricing_href
> FAQ SECTION HREF - $faq_href
> 'BACK' LINK HREF - $back_link_href
> LOGO IMAGE LOCATION - $logo_src


	Strings:

> PAGE TITLE (string) - $title
> WELCOMING TEXT - $welcoming_text
