This document contains essential custom PHP variables used in this project.

	General. Main:

> CURRENT PAGE URL - $url
> CURRENT PAGE URI - $uri

> ROOT DIRECTORY PREFIX (PART OF URL) - $root
> ROOT DIRECTORY PREFIX (LOCAL PATHNAME) - $temp
> MAIN PAGE ADDRESS (URL) - $index

	General. Folders and pathnames:

> CUSTOM SCRIPTS FOLDER ADDRESS (PART OF URL) - $js
> CUSTOM STYLESHEETS FOLDER ADDRESS (PART OF URL) - $stylesheets
> IMAGES FOLDER ADDRESS (PART OF URL) - $imgs
> GENERAL (PUBLIC) PAGES FOLDER ADDRESS (PART OF URL) - $general
> PAGES FOR AUTHORIZED USERS FOLDER ADDRESS (PART OF URL) - $administration

> HEAD TAG FILE ADDRESS (LOCAL PATHNAME) - $head_pathname
> HEADER FILE ADDRESS (LOCAL PATHNAME) - $header_pathname
> SPECIAL HEADER FILE ADDRESS (LOCAL PATHNAME) - $headerAdmin_pathname 
> FOOTER FILE ADDRESS (LOCAL PATHNAME) - $footer_pathname
> 'BACK' LINK FILE ADDRESS (LOCAL PATHNAME) - $backLink_pathname

	General. Scripts and shylesheets:

> CUSTOM SCRIPTS ARRAY (files' names) - $customScripts_array
> CUSTOM STYLESHEETS ARRAY (files' names) - $customStylesheets_array
> CUSTOM STYLES (CSS) - $customStyles_css

	Specific. References (all URL):

> 'BACK' LINK - $backLink_href
> LOGO IMAGE LOCATION - $logo_src
> SPINNER IMAGE LOCATION - $spinner_src
> TEACHERS' APPLICATION FORM PAGE - $appTch_href
> STUDENT APPLICANTS' PAGE - $appStd_href
> STUDENT'S PRIVATE LESSONS APPLICATION FORM PAGE - $appStdPrivate_href
> STUDENT'S GROUP LESSONS APPLICATION FORM PAGE - $appStdGroup_href

  General header references:

> PRE-APPLICATION PAGE - $appMain_href
> ADMINISTRATOR LOG IN PAGE - $authorization_href
> TOP (HEADER) SECTION - $header_href
> CONTACTS SECTION - $contacts_href
> PRICE LIST SECTION - $pricing_href
> FAQ SECTION - $faq_href


  Specific header references:

> APPLICATIONS VIEWING PAGE - $applications_href
> SCHEDULE VIEWING PAGE - $schedule_href
> USER LOG OUT PAGE - $logout_href

	Strings:

> PAGE TITLE (string) - $title
> WELCOMING TEXT - $welcoming_text
> Quote under welcoming - $blockquote_text
> Citation of the quote - $blockquote_citation
> Notes under time schedule - $schedule_note
> Notes under pricing - $pricing_note
> Slogan placed at the bottom of the page - $slogan_text
> Administrator's email address (contacts section) - $email_text
> Administrator's phone number (contacts section) - $phone_text
